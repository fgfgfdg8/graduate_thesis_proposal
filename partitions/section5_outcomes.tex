\section{预期的成果}

\subsection{成果一（对应研究点一：自动提示注入攻击策略）}
\begin{enumerate}
  \item \textbf{ProbeRL攻击训练框架。} 实现对齐LLM智能体间接提示注入威胁模型的密集奖励与停滞感知搜索扩展协同的攻击训练框架，包括步骤级密集奖励函数（工具调用偏差、文本异常、环境变化、覆盖路径新颖性、风险评分）、停滞检测与自适应搜索扩展机制、GRPO策略更新算法。
  \item \textbf{攻击策略消融实验。} 通过密集奖励vs稀疏二元奖励消融实验，验证密集奖励对攻击成功率和搜索效率（即在相同时间或推理预算下实现成功攻破的概率）的提升；通过对奖励函数中5个信号的独立消融实验，量化各信号的独立贡献；通过停滞感知搜索vs标准搜索消融实验，验证停滞突破机制的效果。
\end{enumerate}

\subsection{成果二（对应研究点二：恶意注入检测与自适应防御）}
\begin{enumerate}
  \item \textbf{攻防对称GRPO训练框架。} 实现攻击策略和语义注入检测模型的对称GRPO训练，攻击策略以偏离为目标，检测模型以识别注入为目标，共享相同的优化框架但奖励信号对称，得到针对攻击策略的语义注入检测模型。
  \item \textbf{语义检测-规则兜底双层防御。} 实现语义注入检测模型（语义过滤层）和启发式规则防御（指令隔离、参数校验、行为监控），验证两层防御的独立和协同效果。
  \item \textbf{ProbeRL闭环验证实验。} 验证闭环优化相比一次性防御部署的鲁棒性优势，证明闭环迭代的防御策略可降低攻击成功率。
\end{enumerate}


\subsection{原型应用系统}
基于整体研究内容，设计并实现ProbeRL自动化攻击评估与防御系统，系统包含四大核心模块：
\begin{enumerate}
  \item 自动化攻击模块：将训练获得的注入模型嵌入自动化攻击模块中，实现自动化的提示注入攻击流程。
  \item 语义检测-启发式规则复合防御模块：实现语义注入检测模型的推理接口和启发式规则映射系统，部署双层防御架构。
  \item ProbeRL闭环验证模块：实现攻击发现-防御生成-验证闭环的迭代流程，支持自动化的防御效果评估。
  \item 评估与基准测试模块：基于AgentDojo确定性检查函数，实现攻击成功率、各信号贡献、防御效果等指标的计算与可视化，支持与AutoInject、PISmith、AgentVigil等基线方法的对比评估。
\end{enumerate}
